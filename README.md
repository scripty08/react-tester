# @scripty/react-tester

# Description

react-tester component.

# Usage
```bash
npm install -s @scripty/react-tester
```

##### Client: Example.jsx

```javascript
import React from 'react';
import { ReactTester } from '@scripty/react-tester';

export const Example = () => {

    return (
        <ReactTester />
    );
};
```
